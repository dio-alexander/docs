## API Reference

### Authorizations

Authentication: `Authorization: Bearer <YOUR_API_KEY>`

### Body Parameters

**model** `string` (required)

- Must be `z-image` for this endpoint.

**callBackUrl** `string` (optional)

- URL for webhook notifications.

**input** `object`

- Input parameters for generation.

### Response

- **200**: Success
- **401**: Unauthorized
- **402**: Insufficient Credits

```json
{
  "code": 200,
  "msg": "success",
  "data": { ... }
}
```
