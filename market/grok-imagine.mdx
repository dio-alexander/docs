## API Reference

### Authorizations

Authentication: `Authorization: Bearer <YOUR_API_KEY>`

### Body Parameters

**model** `string` (required)

- Must be `grok-imagine/text-to-image`

**callBackUrl** `string` (optional)

- Webhook URL

**input** `object`

- Image generation parameters

### Response

- **200**: Success
- **401**: Unauthorized

```json
{
  "code": 200,
  "msg": "success",
  "data": { ... }
}
```
