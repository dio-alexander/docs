## Overview

- **True vertical video**: Native Veo 3.1 now fully supports 9:16 output.
- **Global language reach**: Improved non-English success rates.
- **Cost savings**: significantly lower than official pricing.

## API Reference

### Authorizations

Authentication: `Authorization: Bearer <YOUR_API_KEY>`

### Body Parameters

**model** `string` (required)

- `veo3`: Veo 3.1 Quality (text-to-video and image-to-video)
- `veo3_fast`: Veo3.1 Fast generation model

**prompt** `string` (required)
Text prompt describing the desired video content.

- Should be detailed and specific.

**image_urls** `array` (optional)
Image URL list for image-to-video mode.

- 1 image: Video unfolds around this image.
- 2 images: First and last frames.

**generation_type** `string` (optional)

- `TEXT_2_VIDEO`
- `FIRST_AND_LAST_FRAMES_2_VIDEO`
- `REFERENCE_2_VIDEO`

**aspect_ratio** `string` (optional)

- `16:9` (supports 1080P)
- `9:16`
- `Auto`

### Response

- **200**: Success
- **401**: Unauthorized
- **402**: Insufficient Credits

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "task_id": "...",
    "status": "QUEUED"
  }
}
```
