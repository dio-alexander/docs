title: "Extend AI Video"

## Usage Guide

- Extend using original `taskId`.
- Maintain visual consistency.

## Parameter Details

**taskId** `string` (required)

- ID of the video to extend.

**prompt** `string`

- Guide continuation.

## Response

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "taskId": "..."
  }
}
```
