## Upload Method Comparison

### URL File Upload

- No local file required.
- Supports remote resources.
- 30s timeout, max 100MB.

### File Stream Upload

- High efficiency.
- Supports large files.
- Binary transmission.

### Base64 Upload

- Easy integration (JSON).
- Data size +33%.
- Max 10MB.
