## API Reference

### Authorizations

Authentication: `Authorization: Bearer <YOUR_API_KEY>`

### Body Parameters

**model** `string` (required)

- Must be `flux-2/flex-text-to-image`

**callBackUrl** `string` (optional)

- Webhook URL

**input** `object`

- Generation parameters

### Response

- **200**: Success
- **401**: Unauthorized
- **402**: Insufficient Credits

```json
{
  "code": 200,
  "msg": "success",
  "data": { ... }
}
```
