## Usage Guide

- Check current credit balance.
- Monitor usage.

## API Reference

### Authorizations

Authentication: `Authorization: Bearer <YOUR_API_KEY>`

### Response

```json
{
  "code": 200,
  "msg": "success",
  "data": 100
}
```

- **200**: Success
- **401**: Unauthorized
