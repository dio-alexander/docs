## Welcome to Common API

- **[Get Account Credits](common-api/get-account-credits)**: Check your current credit balance.
- **[Get Download URL](common-api/download-url)**: Get direct download URLs for generated files.

## API Overview

### Get Account Credits

`GET /api/v1/chat/credit`

### Get Download URL

`POST /api/v1/common/download-url`
