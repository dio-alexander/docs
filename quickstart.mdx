This guide will help you make your first API call to KIE.

## Step 1: Get Your API Key

1. Sign up or log in at [kie.ai](https://kie.ai)
2. Navigate to the [API Key Management](https://kie.ai/api-key) page
3. Create a new API key and save it securely

<Warning>
Never share your API key or expose it in client-side code.
</Warning>

## Step 2: Make Your First Request

Here's an example of generating an image with the 4o Image API:

<CodeGroup>

```bash cURL
curl -X POST https://api.kie.ai/api/v1/4o/image/generate \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "prompt": "A cute cat playing in a garden",
    "aspectRatio": "1:1"
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.kie.ai/api/v1/4o/image/generate",
    headers={
        "Content-Type": "application/json",
        "Authorization": "Bearer YOUR_API_KEY"
    },
    json={
        "prompt": "A cute cat playing in a garden",
        "aspectRatio": "1:1"
    }
)

data = response.json()
task_id = data["data"]["taskId"]
print(f"Task created: {task_id}")
```

```javascript JavaScript
const response = await fetch("https://api.kie.ai/api/v1/4o/image/generate", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_API_KEY"
  },
  body: JSON.stringify({
    prompt: "A cute cat playing in a garden",
    aspectRatio: "1:1"
  })
});

const data = await response.json();
console.log("Task created:", data.data.taskId);
```

</CodeGroup>

## Step 3: Get the Result

All KIE tasks are asynchronous. You have two options to get results:

### Option A: Poll for Results

Use the record info endpoint to check task status:

<CodeGroup>

```bash cURL
curl -X GET "https://api.kie.ai/api/v1/record-info?taskId=YOUR_TASK_ID" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import time
import requests

def wait_for_result(task_id, api_key):
    while True:
        response = requests.get(
            f"https://api.kie.ai/api/v1/record-info?taskId={task_id}",
            headers={"Authorization": f"Bearer {api_key}"}
        )
        data = response.json()

        if data["data"]["status"] == "completed":
            return data["data"]["result"]
        elif data["data"]["status"] == "failed":
            raise Exception(data["data"]["error"])

        time.sleep(5)  # Wait 5 seconds before polling again
```

</CodeGroup>

### Option B: Use Webhooks (Recommended)

Include a `callBackUrl` in your request to receive results automatically:

```bash
curl -X POST https://api.kie.ai/api/v1/4o/image/generate \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "prompt": "A cute cat playing in a garden",
    "aspectRatio": "1:1",
    "callBackUrl": "https://your-server.com/webhook"
  }'
```

Learn more about webhooks in the [Webhooks Guide](/concepts/webhooks).
