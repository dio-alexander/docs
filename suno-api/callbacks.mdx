## Callback Mechanism Overview

When the generation task is complete or fails, the system sends a POST request to your `callBackUrl`.

### Request Format

```json
{
  "code": 200,
  "msg": "All generated successfully.",
  "data": {
    "callbackType": "complete",
    "task_id": "...",
    "data": [
      {
        "id": "...",
        "audio_url": "...",
        "title": "...",
        "duration": 198.44
      }
    ]
  }
}
```

### Callback Types

- `text`: Text generation complete
- `first`: First track complete
- `complete`: All tracks complete
- `error`: Generation failed

### Response

Return a 200 status code to confirm receipt.
