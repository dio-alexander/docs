## Callback Mechanism Overview

### Request Format

```json
{
  "code": 200,
  "msg": "Modify record generated successfully.",
  "data": {
    "taskId": "...",
    "resultUrls": ["https://..."]
  }
}
```

### Response

Return 200 to confirm receipt.
