---
title: 'Record Info'
description: 'Query task status and results'
---

The Record Info API allows you to check the status of any asynchronous task and retrieve its results.

## Endpoint

```
GET https://api.kie.ai/api/v1/record-info
```

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `taskId` | string | Yes | The task ID to query |

## Request

```bash
curl -X GET "https://api.kie.ai/api/v1/record-info?taskId=task_abc123" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Response

### Pending Task

```json
{
  "code": 200,
  "data": {
    "taskId": "task_abc123",
    "status": "pending",
    "createdAt": "2024-01-15T10:30:00Z"
  }
}
```

### Processing Task

```json
{
  "code": 200,
  "data": {
    "taskId": "task_abc123",
    "status": "processing",
    "progress": 45,
    "createdAt": "2024-01-15T10:30:00Z"
  }
}
```

### Completed Task

```json
{
  "code": 200,
  "data": {
    "taskId": "task_abc123",
    "status": "completed",
    "result": {
      "url": "https://cdn.kie.ai/outputs/...",
      "duration": 120
    },
    "createdAt": "2024-01-15T10:30:00Z",
    "completedAt": "2024-01-15T10:32:00Z"
  }
}
```

### Failed Task

```json
{
  "code": 200,
  "data": {
    "taskId": "task_abc123",
    "status": "failed",
    "error": "Content policy violation",
    "errorCode": "CONTENT_BLOCKED",
    "createdAt": "2024-01-15T10:30:00Z"
  }
}
```

## Task Status Values

| Status | Description |
|--------|-------------|
| `pending` | Task is queued |
| `processing` | Task is being processed |
| `completed` | Task finished successfully |
| `failed` | Task failed |

## Code Examples

<CodeGroup>

```python Python
import requests
import time

def get_task_result(task_id, api_key, timeout=300, poll_interval=5):
    """Poll for task completion with timeout."""
    start_time = time.time()

    while time.time() - start_time < timeout:
        response = requests.get(
            f"https://api.kie.ai/api/v1/record-info?taskId={task_id}",
            headers={"Authorization": f"Bearer {api_key}"}
        )
        data = response.json()["data"]

        if data["status"] == "completed":
            return data["result"]
        elif data["status"] == "failed":
            raise Exception(f"Task failed: {data.get('error')}")

        time.sleep(poll_interval)

    raise TimeoutError("Task did not complete within timeout")

# Usage
result = get_task_result("task_abc123", "YOUR_API_KEY")
print("Result URL:", result["url"])
```

```javascript JavaScript
async function getTaskResult(taskId, apiKey, timeout = 300000, pollInterval = 5000) {
  const startTime = Date.now();

  while (Date.now() - startTime < timeout) {
    const response = await fetch(
      `https://api.kie.ai/api/v1/record-info?taskId=${taskId}`,
      { headers: { "Authorization": `Bearer ${apiKey}` } }
    );
    const { data } = await response.json();

    if (data.status === "completed") {
      return data.result;
    } else if (data.status === "failed") {
      throw new Error(`Task failed: ${data.error}`);
    }

    await new Promise(resolve => setTimeout(resolve, pollInterval));
  }

  throw new Error("Task did not complete within timeout");
}

// Usage
const result = await getTaskResult("task_abc123", "YOUR_API_KEY");
console.log("Result URL:", result.url);
```

</CodeGroup>

## Error Codes

| Code | Description |
|------|-------------|
| 400 | Missing taskId parameter |
| 401 | Unauthorized - invalid API key |
| 404 | Task not found |
