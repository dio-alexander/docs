## Base URL

All API requests should be made to:

```
https://api.kie.ai/api/v1
```

## Authentication

All endpoints require Bearer token authentication:

```bash
Authorization: Bearer YOUR_API_KEY
```

Get your API key from the [API Key Management](https://kie.ai/api-key) page.

## Common Headers

| Header | Value | Required |
|--------|-------|----------|
| `Authorization` | `Bearer YOUR_API_KEY` | Yes |
| `Content-Type` | `application/json` | Yes |

## Response Format

All responses follow this structure:

### Success Response

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    // Response data
  }
}
```

### Error Response

```json
{
  "code": 422,
  "msg": "Validation Error",
  "details": "prompt is required"
}
```

## API Endpoints

### Video Generation

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/veo/generate` | Generate video with Veo 3.1 |
| POST | `/runway/generate` | Generate video with Runway |
| POST | `/kling/generate` | Generate video with Kling |
| POST | `/seedance/generate` | Generate video with Seedance |

### Music Generation

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/generate` | Generate music with Suno |
| POST | `/generate/extend` | Extend existing music |
| POST | `/generate/lyrics` | Generate lyrics |

### Image Generation

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/4o/image/generate` | Generate image with 4o |
| POST | `/4o/image/edit` | Edit image with 4o |
| POST | `/flux/kontext/generate` | Generate with Flux Kontext |
| POST | `/midjourney/generate` | Generate with Midjourney |

### Common

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/record-info` | Get task status and results |

## Rate Limits

Rate limits are applied per API key:

- Requests per minute: Based on plan
- Concurrent tasks: Based on plan
- Daily limits: Based on plan

Check your limits in the [Dashboard](https://kie.ai/dashboard).
